Mengenal Knex library query builder

Raldes Krisnu
Oct 1, 2018 · 2 min read

Hallo semua sekarang kita bahas nih query builder yang lumayan terkenal yaitu adalah KnexJs. apa sih knexjs? knex.js adalah sebuah query builder di javascript yang biasanya digunakan pada nodeJs untuk men-generate database, select, insert dsb. proses generate database ini sangat berguna untuk memanage sebuah project jadi kita tidak perlu membuat database dan tablenya satu persatu atau menginport ulang databasenya. yang kita akan bahas adalah proses generate database dan seeding pada knexjs.

KnexJs ini support beberapa database loh seperti oracle, postgres, mysql, Sqlite3, mariaDB, amazon redshift, mssql. sekarang kita coba aja langsung praktekin. pertama kita install dulu knex nya "npm install knex --save" atau bisa menggunakan global install “npm install knex -g”(saya berasumsi kalo kalian sudah ada project node) kalo sudah kita coba buat sebuah file bernama knexfile.js lalu ikuti seperti contoh dibawah ini :

module.exports = {development: {client: 'postgresql',connection: `postgres://localhost:5432/${process.env.DATABASE_NAME}`,//untuk mengenerate databasemigrations: {directory: __dirname + 'database/migration'},//untuk mengisi default value databaseseeds: {directory: __dirname + 'database/seeds'}}};

code diatas saya menggunakan database postgres. dalam code tersebut pada bagian migrations itu adalah letak file query builder kita untuk mengenerate database, pada bagian seeds adalah untuk seeding value ke database.

sekarang kita buat file migrationnya. buatlah file migrationusers.js atau bisa menggunakan command knex migrate:make namatable lalu akan men-generate sebuah file. lalu copy-paste ke file migrationuser kalian

exports.up = (knex, Promise) => {return knex.schema.createTable('users', (table) => {table.increments('id').primary().unsigned();table.string('username').unique().notNullable();table.string('password');table.timestamp('created_at').defaultTo(knex.raw('now()'));});};exports.down = (knex, Promise) => {return knex.schema.dropTable('users');};

sekarang coba kalian run command knex migrate:latest lalu coba lihat didatabase kalian.

nah sekarang untuk bagian seedingnya knex seed:make seeding kalian bisa lalu copy paste code dibawah ini ke file seeding yang sudah tergenerate.

exports.seed = function(knex, Promise) {// Deletes ALL existing entriesreturn knex('users').del().then(function () {// Inserts seed entriesreturn knex('users').insert([{username: 'Raldes',application_name: 'test123',password:"hash"}]);});};

lalu run command knex seed:run. dan database sudah terisi dengan data diatas.

begitulah kira kira knexJS secara singkat dan beserta contohnya. seperti biasa correct me if i’m wrong atau ada yang mau nambahin dan diskus tentang knexjs dan query builder lainnya silahkan. sekian terima kasih. salam newbie.
